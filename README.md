
## 🏊 javascript 알고리즘 스터디 

- 정해진 날짜에 만나 `문제 목록`의 문제들을 푼다
- 이후 각자의 풀이 방법을 공유한 뒤, 해당 풀이방법 + 개념들을 레파지토리에 PR한다. (템플릿 사용)
  - 각자 이름의 브랜치에서 작업한다.
  - 각자 이름의 폴더에서 작업한다.
  - 커밋 메세지는 문제 이름으로 통일한다. 
  - `solved`, `unsloved`, 등의 라벨을 붙인다.
